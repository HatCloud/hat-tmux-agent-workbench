A personal macOS tmux + AI-agent terminal workbench: agent windows, status-bar task tracking, and workspace snapshots, deployed via `scripts/deploy.sh`.
To deploy, run `scripts/setup agent-guide` for the machine-readable deployment contract (flags, decision points, output schema) and follow it.
Maintainers: read `docs/ARCHITECTURE.md` before changing `agent-tracker/` (Go) or `tmux/`.
